Scribble README
===============

This distribution contains the following information:

- bin

This folder contains the scripts for running the Scribble tooling from the command line.

- docs

This folder contains user and developer guides, as well as javadocs for the Java classes.

- lib

This folder contains the libraries (scribble and third party) that are required to run the tooling.

- samples

Some example scribble protocols and trace files to help you get started.
